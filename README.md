# NotesExample

Это сайт который я накидал за полтора часа. Цель - показать чем примерно мы будем заниматься в этом семестре.

Технологии, использованные в проекте:
1. ASP.NET Core Blazor - как основа
2. Entity Framework Core - для работы с БД
3. SQLite - сама БД
4. Bootstrap - для стилей (чтобы выглядело прилично)

## Развернуть
Клонируем проект и запускаем. БД создаётся автоматически.
